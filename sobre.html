SOBRE O TRABALHO

O objetivo desse trabalho é consumir os dados provenientes de uma API REST e exibir esses dados em uma página com HTML dinâmico e Grafico Google Charts gerado a partir de JavaScript.

Esse projeto contém Mapa, Gráfico Pizza e Tabela. Separados por casos, mortes, suspeitos, recuperados e descartádos, cada um deles por Siglas de cada país e o nome dos estados.

Usando as ferramentas:

  • HTML
  • CSS
  • JavaScript
